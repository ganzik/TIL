# CSS Text 텍스트

## Text Color - 글자 색상

Color 속성은 텍스트의 색상을 지정할 때 사용한다. 컬러는 아래와 같이 지정된다.

- 컬러의 이름 - "red"
- Hex 값 - "ff0000"
- RGB 값 - "rgb(255,0,30)"

### 예

셀렉터를 이용하여 본문(body) 기본 색상은 파란색으로 지정하고 h1 태그는 녹색으로 지정하자

```css
body {
  color: blue;
}

h1 {
  color: green;
}
```
