## 클래스 컨테이너

```html
<div class="container">
  <h2>클래스 container</h2>
  <p>
    div 박스를 컨테이너에 담는다. 가운데 정렬을 하고 화면에 꽉차지 않는다. 아래
    container-flouid 클래스와 비교해보자
  </p>
</div>
```

```html
<div class="container-flouid">
<h2>클래스 container-flouid</h>
<p>화면 꽉 차게 div 박스를 지정한다.</p>
</div>
```
