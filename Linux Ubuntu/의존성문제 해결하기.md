의존성 문제가 생긴 패키지를 삭제한다

![openssh](./imgs/openssh.png)
예시)openssh-server

```bash
apt-get remove openssh-client

apt-get remove openssh-sftp-server

apt-get clean && apt-get purge

apt-get update

apt-get install openssh-server

dpkg -l | grep openssh
```
