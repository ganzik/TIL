# 하드디스크 관리와 사용자별 공간 할당

sata 0(카드슬롯):0(할당넘버) - /dev/sr0

scsi 0(카드슬롯):0(할당넘버) - /dev/sda1

## 파티션 나누기

```bash
# 하드디스크 선택
fdisk /dev/sdb
Command : n     # 새로운 파티션 분할
Select : p     # Primary 파티션 선택
Partition number :     # 파티션 번호 1번 선택(Primary 파티션은 최대 4개까지 생성 가능)
Command : p     # 설정된 내용 확인
Command : w     # 설정 저장
```
