## 네트워크 (Network)

## 네트워크란?

다 수의 컴퓨터(혹은 장비)끼리 데이터를 주고 받기 위해 연결해 놓은 망

## 인터넷(Internet)

internet = inter + network 네트워크의 네트워크  
전 세계 네트워크를 연결하는 최상위 네트워크  
TCP/IP Protocol 기반으로 전 세계 컴퓨터를 하나로 연결

## 인트라넷(Intranet)

독립적인 네트워크
