# Golang

## install

mac 의 homebrew 를 이용해서 간단히 설치한다.

homebrew 가 없는 경우에는 https://golang.org/dl/ 에서 다운로드 받아서 설치한다.

```bash
# 설치
brew install go

# 설치 확인
go version

# go 설치 경로 확인
go env GOROOT
```
