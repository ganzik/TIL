# Background Image 배경 이미지

Internal style sheet  
\<head>태그에 삽입

```html
<style>
  body {
    background-image: url("https://ganzik.github.io/assets/photo/reflection.jpg");
  }
</style>
```

Inline style  
\<body>태그에 삽입

```html
<div
  height="600px"
  width="600px"
  style="background-image: url('https://ganzik.github.io/assets/photo/reflection.jpg')"
></div>
```
